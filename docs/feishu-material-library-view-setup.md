# 飞书素材库视图配置指南

> 本文件为手动配置视图提供精确步骤。所有数据表和字段均已通过 API 创建完毕。

## 一、素材库视图

### 00·素材总览（已存在）

视图类型：网格视图  
筛选：无  
显示字段顺序：
1. 原图片（附件）
2. 视频封面（附件）
3. 原视频（附件）
4. 标题（文本）
5. 平台（单选）
6. 内容形态（单选）
7. 作者（文本）
8. 采集时间（创建时间）
9. 一级分类（文本）
10. 二级分类（文本）
11. 关键词（单选）
12. 情绪标签（单选）
13. 素材价值（单选）
14. 创作状态（单选）
15. 评分（数字）
16. 适合引流（复选框）
17. 使用风险（单选）
18. 人工备注（多行文本）

隐藏字段：图片链接、视频链接、同步状态、视频附件状态、视频下载错误、视频总数、视频成功数、视频失败数、视频处理状态、视频处理进度、视频失败详情、视频任务批次ID、视频失败摘要、平台内容ID、作者账号ID

排序：采集时间（降序）、评分（降序）

### 01·待提炼（需确认视图名）

当前素材库已有视图「10·创作提炼」，功能与规格中的「01·待提炼」一致。

若需要重命名或新建：
- 视图名：01·待提炼
- 筛选：创作状态 = 待筛选 或 创作状态 = 待提炼
- 显示字段：标题、内容描述、原图片、视频封面、关键词、情绪标签、场景标签、爆点分析、素材价值、评分、人工备注

### 02·可创作（需手动创建）

在素材库中手动创建：
1. 点击「+ 新建视图」→ 选择「网格视图」
2. 命名为：02·可创作
3. 点击「筛选」→ 添加条件：
   - 创作状态 = 可创作
   - 或 创作状态 = 创作中
   - 或 创作状态 = 待发布
4. 点击「字段显示」→ 显示以下字段：
   - 标题、原图片、原视频、视频封面
   - 镜头/氛围、爆点分析、命理主题、适合引流
   - 私域价值、素材价值、创作状态、关联创作卡

### 20·视频处理排查（已存在）

视图类型：网格视图  
筛选：视频总数 > 0 且 视频处理状态 != VIDEO_COMPLETE  
显示字段：标题、原始链接、平台、内容形态、视频总数、视频成功数、视频失败数、视频处理状态、视频处理进度、视频失败摘要、视频失败详情、视频任务批次ID、视频附件状态、视频下载错误
